## Текущая задача
**Механики: Универсальная разрушаемость стен (как в Майнкрафте).**



### Контекст задачи
Полная разрушаемость стен: ЛЮБАЯ стена может быть разрушена инструментом. Исключения — только то, что Самосбор не может разрушить: гермостены (`Tex.HERMO_WALL`), гермодвери, лифтовые ячейки (`Cell.LIFT`) и plot-armor объекты.

### Конкретные файлы и паттерны
- **`src/core/world.ts`**: `World.cells` — типизированный массив. `Cell.WALL → Cell.FLOOR` при разрушении.
- **`src/systems/interactions.ts`**: Добавьте обработчик удара по стене: если equipped tool = кирка/отбойник → уменьшить стену HP (новое sparse map `wallHp: Map<number, number>` на World).
- **Version bump**: ОБЯЗАТЕЛЬНО `world.cellVersion++` при разрушении стены (перестройка nav tree).
- **`world.surfaceVersion`**: Тоже обновить (визуал).
- **Неразрушимые**: Только `Cell.LIFT`, `Tex.HERMO_WALL`, `DoorState.HERMETIC_*` и ячейки с plot-armor флагом. ВСЁ остальное разрушаемо — стены квартир, коридоры, перегородки, даже несущие конструкции (это ГИГАХРУЩ, тут нет несущих стен в привычном смысле). `aptMask` НЕ защищает от разрушения — оно только запрещает генераторам переписывать ячейки.
- **Performance**: `wallHp` — sparse Map, не полный типизированный массив. Стена без записи = default HP (100 для обычных стен, 200 для металлических `Tex.METAL`).
- **Gameplay**: Разрушение стены создаёт обломки на полу (cosmetic surface mark `'rubble'`), шум привлекает монстров в радиусе 30 клеток (`publishEvent('wall_destroyed', ...)`), и открывает потенциальные шорткаты / секретные проходы. Это геймплейно значимо!
- **Самосбор**: Стены, разрушенные игроком, НЕ восстанавливаются при Самосборе — Самосбор перестраивает только свою зону. Разрушенные стены вне зоны перестройки остаются.
