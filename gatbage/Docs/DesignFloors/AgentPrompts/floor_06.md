Make the floor a living municipal archive: shelf canyons, record islands, clerk counters, sealed aisles, document elevators, reading pits and administrative cul-de-sacs.

## Tasks

1. Build a shelf-canyon macro layout with long stacks crossed by occasional clerk bridges and archive gates.
2. Add loops where the player can circle around blocked shelves instead of backtracking through one corridor.
3. Create at least three motifs: stack canyon, counter hall, sealed record vault, reading pit, archive service lift.
4. Make identity/access decisions physical: steal through stacks, queue at counters, force a vault, expose a record.
5. Keep named archive NPC/content rooms protected and reachable.
6. Run `npm run check`.
