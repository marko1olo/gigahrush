# ГИГАХРУЩ: МЕГА-АПДЕЙТ "100 АГЕНТОВ"

Настоящий план содержит 100 изолированных, но взаимосвязанных задач для 100 независимых агентов (Jules). Каждая задача направлена на реализацию масштабного обновления игры, затрагивающего сюжет, генерацию, ИИ, боевую систему, рендер и мобильную платформу.

## ⚠️ ОБЩИЕ ПРАВИЛА ДЛЯ ВСЕХ АГЕНТОВ (AGENTS.md)
1. **Никакого хардкода и костылей.** Все системы должны быть универсальными, расширяемыми и опираться на данные (`src/data/`).
2. **Соблюдение архитектуры слоев.** `core/` (типы) -> `data/` (справочники) -> `gen/` (генерация) -> `systems/` (логика) -> `render/` (отрисовка). Логики геймплея в рендере быть не должно!
3. **Оптимизация (Iron Law).** Никакого поиска пути O(W²) или перестроения навигационного графа в реальном времени во время активной симуляции. 
4. **Скромность в зависимостях.** Новые npm-пакеты, физические движки или UI-фреймворки ЗАПРЕЩЕНЫ. Используем существующий стек (Canvas, WebGL, Vanilla TS).
5. **Тестирование.** Проверяйте изменения командами `npm run check:readonly` или `npm run check:full`.
