## Текущая задача
**Подготовка структуры 'Харизматичный злодей и герой'. Специальные НПЦ-профили для использования в синематиках.**



### Контекст задачи
Создать структуры данных для двух уникальных НПЦ — «Злодея» и «Героя» — которые будут фигурировать в синематиках (`CameraMode: 'trailer'` из `src/systems/camera.ts`).

### Конкретные файлы и паттерны
- **`src/data/npc_plot_packages.ts`**: Зарегистрируйте два новых пакета через `MAIN_PLOT_NPC_PACKAGES`. Злодей: высокий уровень, PSI-оружие, фракция `Faction.CULTIST` или `Faction.WILD`. Герой: фракция `Faction.LIQUIDATOR`, хорошее оружие.
- **`src/data/plot.ts`**: Через `registerNpcPackageFromPlotNpc()` зарегистрируйте обоих. Дайте им `homeFloorKey` для начального размещения.
- **`src/entities/npc_visuals.ts`**: Если нужен уникальный визуал — добавьте `npcVisualId` в пакет. Просмотрите существующие визуальные семейства.
- **Синематики**: Данные для синематик-режиссёра (marx_82-85) будут ссылаться на `plotNpcId` этих персонажей строково.
