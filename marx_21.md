## Текущая задача
**База Ликвидаторов: Геометрия Арены (трибуны, ринг в центре).**



### Контекст задачи
В центре Базы Ликвидаторов — арена (ринг) с трибунами. Это большая зона, где проходят бои.

### Конкретные файлы и паттерны
- **Генератор** (`liquidator_base.ts`): Создайте комнату `arena` ~50x50 клеток. Стены — `Tex.METAL`, пол — `Tex.F_CONCRETE`. Вокруг ринга поставьте ряды `Feature.CHAIR` (трибуны).
- **Центральный ринг**: Прямоугольная область ~20x20 без фич, только пол. Ограждение — ряд `Feature.TABLE` или кастомные walls с низкой текстурой.
- **`world.roomMap`**: Арена — отдельная `Room` с `name: 'Арена'`, `type: RoomType.COMMON`.
- **Двери**: Два входа на арену (`Cell.DOOR`), ведущие из коридора.
- **Важно**: Этот генератор будет переиспользоваться системой ставок (marx_25-27) через roomId арены.
