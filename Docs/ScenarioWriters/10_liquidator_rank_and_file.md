# Сценарист 10: Рядовые ликвидаторы

## Архетип

Рядовые ликвидаторы - не плакатные герои. Это люди в противогазах, с плохим снаряжением, приказом и шансом не вернуться. Они одновременно пугают гражданских и сами боятся сирены.

## Тон

Командный, грубый, усталый. Речь через операции: зачистить, вынести, опечатать, отойти, фильтр, грабли, пена, ведро, журнал поражения. Черный юмор допустим, но без героического блеска.

## Целевые файлы

- `src/data/dialogue.ts`: `FACTION_LINES` для `Faction.LIQUIDATOR`.
- `src/systems/ai/barks.ts`: liquidator combat/ambient lines.
- `src/data/contracts.ts`: ликвидаторские задания.
- `src/data/rumors.ts`: слухи про зачистки, пенобетон, пропавших.
- `src/data/notes.ts`: акты зачистки.

## Что переписать

1. Дать ликвидаторам несколько уровней: новобранец с граблями, линейный боец, санитар, усталый сержант.
2. Избегать "мы защитим народ". Вместо этого: "дверь заварить", "фрагменты в ведро", "фильтр мокрый", "приказ не обсуждать".
3. Слухи от гражданских о ликвидаторах должны быть амбивалентны: спасли, сожгли, забрали, не объяснили.
4. Контракты должны звучать как приказ/наряд.

## Словарь

"зачистка", "опечатать", "ведро", "фрагменты", "грабли", "пенобетон", "РБХОПЗ", "фильтр", "сектор", "самоликвидация" осторожно.

## Примеры строк

- "Не геройствуй рядом с нами. Потом в рапорте некуда вписывать твою инициативу."
- "Черную слизь граблями, зеленую не трогать, белую вообще не называй без старшего."
- "Дверь заварили не потому, что злые. Потому что изнутри уже отвечали не жильцы."
- "Фильтр намок - отходишь. Спорить будешь, когда будет чем дышать."
- "После отбоя сначала считаем своих. Потом чужих. Потом то, что уже без разницы."

## Игровая задача

Ликвидаторы должны давить на игрока как порядок с плохими методами. С ними можно сотрудничать, бояться их, обманывать, но нельзя воспринимать их как чистых рыцарей.

## DoD

- 30+ строк для ликвидаторского слоя.
- Есть различие между рядовым, санитаром, сержантом.
- Нет романтизации зачистки.
- После правки `npm run typecheck`.
