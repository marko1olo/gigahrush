# Сценарист 23: Рабочие промзоны

## Архетип

Рабочие, мастера смены, контролеры качества, грузчики, станочники. Их голос нужен для промзоны концентрата, производственного пояса и любых factory/production текстов.

## Тон

Смена, шум, брак, травма, норма, усталость, матерная энергия без прямого мата. Рабочий не философствует о системе; он говорит, что линия жует рукав и план всё равно требуют.

## Целевые файлы

- `src/data/dialogue.ts`: mechanic/turner/electrician/cook overlaps.
- `src/data/contracts.ts`: repair, defend, supply, sabotage, retrieve.
- `src/data/rumors.ts`: factory, production, shortage.
- `src/data/notes.ts`: сменные журналы, акты брака.
- `src/data/factories.ts`, `src/systems/production.ts`: только если есть texts.
- `src/gen/design_floors/production_belt.ts` if present.

## Что переписать

1. Дать рабочим разные роли: мастер, усталый сменщик, новый парень, контролер качества, грузчик.
2. Производство должно звучать как horror через процесс: линия шевелится, брак пахнет, план не отменяют, душевые пустые.
3. Контракты: починить, остановить, запустить с браком, украсть партию, защитить смену.
4. Связать с концентратом, фильтрами, дверями, патронами.

## Словарь

"смена", "линия", "норма", "брак", "мастер", "станок", "план", "рукав", "партия", "акты".

## Примеры строк

- "Линия стала жевать медленнее. Мастер сказал: значит, думает."
- "План не сняли даже после сирены. Только фамилии с доски сняли."
- "Брак пахнет сладко. Хорошая партия так не пахнет."
- "Если станок молчит, не радуйся. Он слушает."
- "Душевые пустые, а вода идёт. Значит, смена где-то не там."

## Игровая задача

Рабочие тексты должны превратить production в живую, опасную систему с людьми и решениями, а не таблицу output.

## DoD

- 30+ factory lines.
- Минимум 8 строк с конкретными production hooks.
- Нет абстрактной критики "системы"; только работа, тело, план, риск.
- После правки `npm run typecheck`; systems changes - `npm run check`.
