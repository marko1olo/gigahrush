# Сценарист 46: Ад, Underhell и мясные этажи

## Поверхность

Hell/Underhell - не фэнтезийный ад. Это мясной, культовый, промышленный, мокрый низ ГИГАХРУЩА: пороги, цеха, кожа стен, Вестники, обожженные, cult logistics.

## Тон

Мясо без красивой готики. Запах, тепло стены, крюк, ведро, ожог, порог, мокрый пол, свеча, голоса из вентиляции. Страшно должно быть телесно и бытово.

## Целевые файлы

- `src/gen/hell/*`
- `src/data/plot.ts`: Hell-related NPCs.
- `src/data/rumors.ts`: Hell/Herald/Matka topics.
- `src/data/notes.ts`: адские записки.
- `src/data/samosbor_variants.ts`: meat variant.
- `src/entities/herald.ts`, `src/entities/matka.ts`: descriptions if player-facing.

## Что переписать

1. Убрать "преисподняя как эпос". Это этаж, где слово "этаж" уже не работает.
2. Вестники - сторожа заглушки/порога, не ангелы и не боссы из легенды.
3. Мясной самосбор: не "кровавый ужас", а стены теплеют, тихая комната врет, пол липнет.
4. Культ здесь должен быть практичнее: свечи, охрана, порог, списки, жертвенные долги.

## Словарь

"тепло", "мокрый", "крюк", "порог", "ожог", "ведро", "шов", "мясо", "сторож", "не называй".

## Примеры строк

- "Стена теплая не потому, что живая. Потому что рядом кто-то перестал быть холодным."
- "Вестник не молится. Он стоит, пока проход делает вид, что закрыт."
- "Мясной сбор тихий сначала. Кухня тоже тихая, когда в ней никого не осталось."
- "Крюк дёрнулся в руке. Значит, за стеной ещё есть вес."
- "Underhell ниже не по этажам. Ниже - это когда назад уже не тот глагол."

## Игровая задача

Hell/Underhell должны быть поздними high-risk floors с телесным horror, культом и конкретными квестами.

## DoD

- 30+ hell/meat lines.
- Нет fantasy-адского пафоса.
- Есть practical warnings and quest hooks.
- После gen/system changes `npm run check`.
