# Сценарист 22: Долговая охрана этажа 69

## Архетип

Охранник, вышибала, долговой бухгалтер, администратор, смотрящий за комнатами. Эти люди делают этаж 69 опасным без explicit-контента: контроль дверей, долги, рейды, компромат, подкуп.

## Тон

Низкий, деловой, угрожающий. Не бандитский цирк, а спокойная фраза человека, который знает, какая дверь заперта и где лежит твоя бумага.

## Целевые файлы

- `src/data/contracts.ts`: debt/blackmail/raid tasks.
- `src/data/rumors.ts`: floor 69 debt rumors.
- `src/data/notes.ts`: ledger, receipts, raid warnings.
- `src/gen/design_floors/floor_69.ts` if present.
- `src/data/dialogue.ts`: guard/storekeeper styles if added.

## Что переписать

1. Создать долговой язык: строка в книге, процент, комната, охрана, услуга, "не вовремя".
2. Рейды Министерства и ликвидаторов должны менять текст: кто предупреждает, кто сдает, кто закрывает людей в комнатах.
3. Охранник не должен говорить как Барни. Он не фронтовик, он хозяин узкого коридора.
4. Долговые записки должны давать gameplay: выкупить, подделать, украсть, сжечь, продать.

## Словарь

"книга", "строка", "процент", "ключ", "рейд", "заклад", "комната", "черный вход", "свидетель", "погасить".

## Примеры строк

- "В книге ты легче, чем в коридоре. В коридоре спорят, в книге платят."
- "Ключ дам. Дверь потом сама спросит, кто тебя пустил."
- "Рейд любит шумных. Тихие платят заранее."
- "Строку можно сжечь. Долг дымом не считается, но люди верят, пока темно."
- "Охрана не спасает. Охрана решает, кого не трогать первым."

## Игровая задача

Долговая охрана должна дать floor 69 механическую основу: debt/trust/heat, доступы, сделки, угрозы и последствия.

## DoD

- 20+ debt/guard/admin lines.
- Есть 5+ quest hooks.
- Нет романтизации криминала; угроза бытовая и деловая.
- После правки `npm run typecheck`.
