# Сценарист 30: Охотники и сталкеры вылазок

## Архетип

Охотники, проводники, сталкеры подъезда, одиночки с картой, люди, которые ходят за лутом дальше безопасной зоны. Барни и Громный не входят сюда; это независимые вылазочные голоса.

## Тон

Практичный, немногословный, суеверный от опыта. Они говорят через маршруты, приметы, патроны, метки, контейнеры и цену лишнего килограмма.

## Целевые файлы

- `src/data/dialogue.ts`: `Occupation.HUNTER`, `Occupation.TRAVELER`.
- `src/data/contracts.ts`: expedition, retrieve, kill, inspect.
- `src/data/rumors.ts`: floors, rare items, monsters, containers.
- `src/data/notes.ts`: карты, зарубки, маршрутные записки.
- `src/systems/route_cues.ts`: if player-facing route text exists.

## Что переписать

1. Сделать вылазки центром речи: готовь воду, фильтр, патроны, маршрут назад.
2. Охотники должны давать counterplay clues, но не как tutorial. "Не стой в углу", "глаз любит лампу".
3. Добавить route folklore: метки мелом, запахи лифта, сухие батареи, шум стен.
4. Не писать их как романтичных одиночек. Они торгуются и боятся.

## Словарь

"маршрут", "метка", "вода", "фильтр", "магазин", "контейнер", "назад", "короткий путь", "тварь", "примета".

## Примеры строк

- "Маршрут без обратной стрелки - это не маршрут, а просьба к стене."
- "Глаз любит лампу. Лампу не любишь - живее будешь."
- "Лишняя банка весит меньше, чем жажда на обратном пути."
- "Если контейнер сухой после мокрого самосбора, не открывай первым."
- "Не верь короткому пути, если он появился после писка."

## Игровая задача

Охотники должны поддерживать core loop: lead, preparation, travel, decision, return.

## DoD

- 30+ expedition/hunter lines.
- Минимум 10 строк с practical gameplay advice.
- Нет романтики "вольного сталкера"; есть риск и цена.
- После правки `npm run typecheck`.
