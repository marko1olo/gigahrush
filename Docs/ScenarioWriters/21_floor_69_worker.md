# Сценарист 21: Проститутка с этажа 69

## Архетип

Взрослая работница этажа 69: проститутка, артистка, проводница по долгам и чужим секретам. Не делать ее декорацией или фансервисом. Она человек с работой, страхом, клиентами, охраной, долгом, аптечкой и планом на случай рейда.

## Тон

Уличный, усталый, умный, прямой. Она может быть ласковой как прием, но речь всегда держит границы. Сексуальные детали не описывать. В центре - деньги, безопасность, документы, шантаж, защита, медицина, выход.

## Целевые файлы

- `Docs/DesignFloors/floor_69.md` как safety contract.
- `src/gen/design_floors/floor_69.ts` if present.
- `src/data/rumors.ts`: floor 69 rumors.
- `src/data/contracts.ts`: hide/protect/expose/pay debt/medical supply.
- `src/data/notes.ts`: debt slips, clinic notes, blackmail notes.
- `src/data/dialogue.ts`: adult worker generic lines if added.

## Что переписать

1. Все тексты этажа 69 держать adult-only и не explicit.
2. Создать голос одной работницы: не "роковая женщина", а практичная взрослая, которая видит людей насквозь, потому что иначе не выжить.
3. Основные решения: защитить, вывести, заплатить долг, украсть запись, сдать чиновника, спрятать от рейда, принести лекарства.
4. Не использовать унизительные описания тела. Ужас - в долге, рейде, охране, шантаже и самосборе за запертой дверью.

## Словарь

"долг", "комната", "рейд", "запись", "клиника", "охранник", "клиент", "граница", "документ", "наличные", "не при всех".

## Примеры строк

- "Милый, смотреть можно на ценник. На дверь тоже смотри: рейд ходит тише клиентов."
- "Долг тут липнет хуже слизи. Слизь хоть видно."
- "Запись не продавай первому чиновнику. Второй даст больше и меньше вопросов."
- "Если сирена начнется, комнату закрою. Но бесплатно я только воздухом делюсь, и то не всегда."
- "Мне не спасатель нужен. Мне нужен человек, который умеет молчать у нужной двери."

## Игровая задача

Этаж 69 должен дать социальный noir-horror: долг, рейд, шантаж, медицина, защита. Игрок должен делать моральный выбор, а не получать пикантный текст.

## DoD

- Все персонажи и контекст явно взрослые.
- Нет explicit sex prose.
- 20+ строк работницы с границами, долгом и угрозой.
- После правки `npm run typecheck`; при floor gen - `npm run check`.
