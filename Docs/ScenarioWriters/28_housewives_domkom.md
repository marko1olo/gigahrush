# Сценарист 28: Домком, хозяйки и старшие секций

## Архетип

Домком, старшая по секции, хозяйки, люди с ключами от общего ящика, контролем очереди и моральной властью над соседями. Это бытовая власть снизу.

## Тон

Строгий подъездный реализм. Забота через контроль: кто взял воду, кто не сдал свечу, кто стучал, кто притащил грязь, кто чужой.

## Целевые файлы

- `src/data/dialogue.ts`: `Occupation.HOUSEWIFE`.
- `src/data/context_lines.ts`: theft, repeated help, container, safe zone.
- `src/data/rumors.ts`: shelter tallies, supply caches, theft.
- `src/data/notes.ts`: объявления, списки, жалобы.
- `src/systems/shelter_tally.ts`: only player-facing lines if present.

## Что переписать

1. Домком должен звучать не как Министерство. Это соседская власть, а не кабинет.
2. Добавить реплики про общие запасы, свечи Истотита, воду, гермодвери, детей, чужаков.
3. Слухи после краж и помощи должны идти через домкомовский язык: "видели", "записали", "спросим".
4. Не делать всех хозяйк добрыми или злыми; они практичны.

## Словарь

"список", "общий ящик", "ключ", "соседи", "свеча", "ведро", "кому положено", "записала", "после отбоя", "не шуметь".

## Примеры строк

- "Воду из общего ящика берут по списку, а не по жажде."
- "Ты у гермы не стой красивый. Или заходи, или людям не мешай бояться."
- "Свечу верни. Истотит не церковь, но темнота одинаковая."
- "Я записала, кто стучал после отбоя. Не для доноса. Пока."
- "Хлеб положи на стол, руки покажи, потом поговорим."

## Игровая задача

Домкомы должны создавать социальные последствия: доверие, кражи, укрытие, доступ к общим запасам, доносы.

## DoD

- 25+ housewife/domkom lines.
- Минимум 8 строк связаны с контейнерами/общими запасами.
- Голос не совпадает с Ольгой и Министерством.
- После правки `npm run typecheck`.
