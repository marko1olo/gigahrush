# Сценарист 34: Тихая соседка

## Архетип

Тихая соседка - отдельный сценарный архетип внешней ячейки: женщина из жилой зоны, которая выглядит как помощь, а на деле держит маршруты, хлеб, черную ладонь и чужие обязательства.

## Тон

Мягкий, домашний, опасный. Она говорит почти как заботливая соседка, но фразы имеют второе дно. Отличить от Ольги: Ольга заботится без вербовки, тихая соседка всегда оставляет крючок.

## Целевые файлы

- `src/data/rumors.ts`: `lead_living_external_cell_neighbor`, outcome rumors.
- `src/data/notes.ts`: календарь, квитанции, коробка.
- `src/data/dialogue.ts`: citizen/cult overlap if generic line added.
- `src/data/contracts.ts`: route, trust, betray, expose.
- `src/gen/living/*`: related NPC/room if present.

## Что переписать

1. Дать ей 12-16 реплик: забота, проверка, намек, просьба, мягкая угроза.
2. Она не произносит "Чернобог" первой. Она говорит "свои", "маршрут", "не при всех".
3. Решения игрока должны менять слухи: доверился, сдал, продал, украл.
4. Не делать ее очевидной злодейкой.

## Словарь

"чай", "хлеб", "календарь", "коробка", "маршрут", "свои", "тише", "не при всех", "долг", "потом".

## Примеры строк

- "Хлеб бери. Пустые руки у двери хуже пустого живота."
- "За календарем не смотри при свете. Соседи свет любят больше правды."
- "Маршрут дам, если пойдешь тихо и вернешься без лишних вопросов."
- "Барни хороший человек. Поэтому ему не всё надо знать сразу."
- "Свои не спрашивают, почему помогли. Они потом понимают."

## Игровая задача

Тихая соседка должна быть морально сложной точкой: помощь с долгом, культ без балахона, route access с риском.

## DoD

- 12+ direct character lines.
- Outcome слухи отражают player choice.
- Отличима от Ольги и домкома.
- После правки `npm run typecheck`.
