# Сценарист 25: Лифтеры, метро и проводники

## Архетип

Лифтеры, дежурные платформ, машинисты темной пересадки, люди, которые знают маршруты, но не доверяют им. Лифты и метро - не транспорт, а рискованный договор с пространством.

## Тон

Транспортный, усталый, суеверно-практичный. Билеты, кнопки, платформы, остановки, запах другого этажа, поезд без пассажиров, теплый лифт.

## Целевые файлы

- `src/data/metro.ts`
- `src/data/rumors.ts`: lift/metro route rumors.
- `src/data/notes.ts`: билеты, схемы, предупреждения.
- `src/data/contracts.ts`: escort/deliver/inspect across floors.
- `src/systems/rail_trains.ts`, `src/systems/lift_arachna.ts`: only player-facing messages.
- `src/gen/design_floors/dark_metro.ts` if present.

## Что переписать

1. Добавить голоса транспорта: дежурный, машинист, лифтер, пассажир, застрявший.
2. Подсказки по маршрутам сделать diegetic: теплые кнопки, запах, неправильная станция, билет с зубами.
3. Метро и лифты должны давать выбор: ехать, ждать, выйти, спрятаться, рискнуть коротким путем.
4. Не писать "пространственный портал"; писать "лифт привез запах и отказался ехать обратно".

## Словарь

"кнопка", "кабина", "платформа", "билет", "станция", "рельсы", "двери закрываются", "теплый", "запах", "не выходи".

## Примеры строк

- "Кнопка теплая. Кто-то уехал недавно или ещё не доехал."
- "Поезд пустой не потому, что безопасный. Просто он уже всех высадил."
- "Билет прокомпостирован зубами. Такой иногда проходит, если не показывать свету."
- "Если станция повторилась, сиди. Второй раз она обычно проверяет смелых."
- "Двери закрываются. Руки убрать, мысли тоже."

## Игровая задача

Транспортные тексты должны помогать ориентироваться и усиливать страх перемещения между этажами.

## DoD

- 25+ lift/metro lines.
- Минимум 8 строк дают route cues.
- Нет техно-магического пафоса.
- После правки `npm run typecheck`; systems/route changes - `npm run check`.
