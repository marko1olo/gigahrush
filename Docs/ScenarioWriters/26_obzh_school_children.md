# Сценарист 26: Школа ОБЖ и дети

## Архетип

Дети, учителя ОБЖ, школьный журнал эвакуации, убежище, взрослые, которые пытаются отвлечь детей от гермодвери. Писать осторожно: дети усиливают ужас через ответственность, не через эксплуатацию страдания.

## Тон

Простой, сдержанный, тревожный. Детские реплики могут быть буквальными и странно точными. Учитель говорит устало и строго.

## Целевые файлы

- `src/data/rumors.ts`: `room_obzh_school_gap`, `samosbor_obzh_shelter`, player actions.
- `src/data/notes.ts`: школьные инструкции, журналы.
- `src/data/dialogue.ts`: `Occupation.CHILD` lines.
- `src/data/contracts.ts`: escort/protect/repair shelter tasks.
- `src/gen/living/*obzh*` if present.

## Что переписать

1. Не делать детей "жуткими куклами". Они обычные, поэтому страшнее.
2. ОБЖ должно быть gameplay: закрыть проем, принести door kit, отвести группу, отвлечь от голоса.
3. Учитель/воспитатель говорит практично: "считаем головы", "не смотрим в глазок", "держим руки занятыми".
4. Реплики детей короткие, конкретные, иногда с неправильным пониманием.

## Словарь

"журнал", "головы", "дверь", "глазок", "мел", "карта детей", "не отвечай", "урок", "убежище", "комплект двери".

## Примеры строк

- "Нам сказали, если мама зовёт из двери, надо считать до ста и не отвечать."
- "У Вити мел сам пишет стрелки. Учительница забрала, но стрелки остались."
- "Дети от двери, руки на стол. Кто плачет - плачет тихо, это тоже работа."
- "Журнал эвакуации не врёт. Просто иногда имена в нём появляются раньше детей."
- "Комплект двери не трогай без взрослого. Взрослый сегодня ты."

## Игровая задача

Школа ОБЖ должна давать сильные protect/repair/escort решения без дешевого шока.

## DoD

- 20+ school/child/teacher lines.
- Нет жестоких подробностей ради эффекта.
- Есть минимум 3 playable decisions: repair, escort, distract/protect.
- После правки `npm run typecheck`; gen changes - `npm run check`.
