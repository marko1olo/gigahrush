# Сценарист 32: Черная ладонь и внешняя ячейка

## Архетип

Внешняя ячейка культа: соседка с коробкой за календарем, черная ладонь на стене, маршрутные квитанции, хлеб и дым как вербовка. Это скрытый социальный культ, не парад балахонов.

## Тон

Тихий, соседский, конспиративный. Угроза через доброту с условием: дали хлеб, теперь знают твой долг; помогли пройти, теперь ждут услугу.

## Целевые файлы

- `src/data/rumors.ts`: `faction_chernobog_recruitment`, `lead_living_external_cell_neighbor`, related player actions.
- `src/data/notes.ts`: ладонь, маршрут, квитанции.
- `src/data/contracts.ts`: contraband, betrayal, report, trust.
- `src/gen/living/*`: тихая соседка/external cell content if present.
- `src/data/dialogue.ts`: cult/citizen faction overlap.

## Что переписать

1. Развить черную ладонь как знак адреса и долга, не просто "символ зла".
2. Соседские культисты должны звучать нормально до последней фразы.
3. Игроку дать решения: довериться, взять маршрут, сдать Барни, продать Ваньке, промолчать.
4. Не объяснять культ напрямую. Пусть говорят коробки, квитанции, хлеб, календарь.

## Словарь

"ладонь", "календарь", "квитанция", "маршрут", "хлеб", "дым", "не при соседях", "долг", "тихая дверь", "свои".

## Примеры строк

- "Ладонь за календарем не для красоты. Это чтобы свои не стучали громко."
- "Хлеб возьми. Потом решишь, услуга это была или долг."
- "Маршрут сухой до второй трубы. Дальше не молись, считай шаги."
- "Барни любит такие бумажки. Потому и не показывай ему сразу."
- "Чернобог тут ни при чем, милый. Просто кто-то должен помнить, кому ты обязан."

## Игровая задача

Внешняя ячейка должна давать stealth/social gameplay: доверие, контрабанда, донос, предательство, скрытый доступ.

## DoD

- 20+ external-cell lines.
- Есть минимум 5 outcomes reflected in rumor/notes.
- Культ звучит соседски, не театрально.
- После правки `npm run typecheck`.
