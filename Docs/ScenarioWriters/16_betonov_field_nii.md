# Сценарист 16: Бетонов и полевые НИИ-экспедиции

## Архетип

Полевые экспедиции НИИ, Бетонов, лаборанты с рюкзаками, маршруты в Коллекторы, образцы после Z+22. Это не кабинетная наука, а научная вылазка с фильтром, пустой тарой и шансом не вернуться.

## Тон

Экспедиционный, сухой, с усталостью stalker-like. Записки должны быть практическими и немного подлыми: кто продал карту, кто оставил сменщика у шкафа, кто назвал трофей образцом ради премии.

## Целевые файлы

- `src/data/notes.ts`: уже есть блок "Экспедиция Бетонова / НИИ".
- `src/data/rumors.ts`: `faction_scientist_notes` and related leads.
- `src/data/contracts.ts`: field sample, escort, retrieve.
- `src/data/pneumomail.ts`: письма экспедиций.
- `src/gen/maintenance/*`: если есть POI.

## Что переписать

1. Расширить полевые записки до полноценного набора: карта, приказ, срыв маршрута, предательство, неверный лифт, образец, продажа Министерству/рынку/Якову.
2. Язык должен быть "полевой приказ", а не "древние тайны".
3. Добавить решения: сдать НИИ, продать рынку, отдать Министерству, вскрыть самому, сжечь.
4. Связать с водой, фильтрами, йодом, тарой, Z-метками.

## Словарь

"маршрут", "Z+22", "проба", "тара", "фильтр", "йод", "лифт", "полевой приказ", "не вскрывать", "сдать под подпись".

## Примеры строк

- "До Коллекторов взять воду, фильтр и пустую тару. Кто взял только смелость, тот уже в списке."
- "Бетонов сказал: образец не трофей. Потом продал трофей как образец."
- "Лифт после Z+22 пах йодом. Вышли все. Вернулись две подписи."
- "Черную сжечь, белую не смотреть, серебристую не показывать рынку при свете."
- "Если проба просит открыть крышку, записать просьбу и отойти."

## Игровая задача

Полевые НИИ-тексты должны создавать вылазки, а не фон. Игрок получает маршрут, риск, получателя и моральную цену образца.

## DoD

- 20+ expedition lines.
- Есть минимум 4 разных получателя/исхода.
- Строки читаются как полевой документ или слух, не как лор-эссе.
- После правки `npm run typecheck`.
