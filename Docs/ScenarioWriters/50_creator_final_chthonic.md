# Сценарист 50: Творец

## Персонаж/сущность

Творец - финальная хтоническая сущность, нечеловеческий разум, должность без таблички. Он не злой бог, не демон, не финальный монологер. Он ближе к маронарию: зеленое доказательство, повтор двери, логика без человека, разум, который не разговаривает, а пересчитывает условия.

## Тон

Лавкрафтовская нечеловечность без лилового пафоса. Страх через несовместимость мышления: Творец не угрожает, не злится, не убеждает. Он считает, исправляет, доказывает, стирает свидетеля как ошибку в ведомости. Игрок для него - не герой, а переменная, которая прошла не туда.

## Целевые файлы

- `src/entities/creator.ts`
- `src/data/plot.ts`: `void_warning`, final quest lines.
- `src/data/void_protocols.ts`
- `src/data/notes.ts`: VOID/Creator notes.
- `src/data/rumors.ts`: late VOID rumors.
- `src/gen/void/*`
- `src/systems/death.ts`, `src/systems/world_log.ts`: only if final messages live there.

## Что переписать

1. Убрать любые речи Творца как человека. Если есть lines, сделать их протоколами, сбоями, повторяющимися фрагментами, UI/log anomalies.
2. Маронарийный вайб: зеленый свет, повтор двери, "доказано", "не требуется", "ошибка свидетеля", "условие закрыто".
3. Лавкрафт тут не "древний космос", а невозможность человеческой категории: комната не комната, память не владелец, дверь не выход, фамилия не имя.
4. Финальная задача должна оставаться играбельной: убить/избежать/оставить шип/вернуться. Текст не должен скрыть objective.

## Словарь

"условие", "доказано", "повтор", "свидетель", "зелёный", "контур", "исправлено", "не требуется", "переменная", "дверь совпала", "память снята".

Запрет: "я бог", "взгляни на истину", "смертный", "вечность", "мироздание", "хаос".

## Примеры строк

- "УСЛОВИЕ: свидетель обнаружил контур. ИСПРАВЛЕНИЕ: свидетель не требуется."
- "Дверь совпала с дверью. Вывод: выход ошибочен."
- "Зелёный источник не светит. Он доказывает, что смотреть было лишним."
- "Память снята с учета. Фамилия оставлена для повторной проверки."
- "Творец не заметил тебя. Хуже: он учел тебя."

## Игровая задача

Финал должен стать нечеловеческим, но не туманным для игрока. Игрок понимает, что делать, но не получает человеческого объяснения, зачем Творец существует.

## DoD

- Final/VOID text has nonhuman protocol voice.
- No villain speech.
- Objective clarity preserved.
