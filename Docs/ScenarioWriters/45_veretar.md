# Сценарист 45: Веретар

## Поверхность

Веретар - белое окно, область вместо комнаты, засвеченная фотография, сухой песок, неправильный двор. Это не "светлая бездна", а чужой наружный контур, куда нельзя смотреть.

## Тон

Белый свет, сухость, окно, фото, песок, занавеска, карта, свидетель. Холодная бытовая опасность: завесь, оттащи человека, замажь щель, не экономь путь ценой свидетеля.

## Целевые файлы

- `src/data/samosbor_variants.ts`
- `src/data/samosbor_director.ts`
- `src/systems/samosbor.ts`
- `src/data/rumors.ts`: veretar topics.
- `src/data/notes.ts`: overexposed photos, sand, window.
- `src/data/items.ts`: veretar_sand/overexposed_photo descriptions if present.

## Что переписать

1. Веретарские строки должны быть визуально простыми: белое, сухое, окно, занавеска.
2. Убрать любые "сияние истины". Белый свет - дефект, риск, чужая область.
3. Основные решения: завесить, оттащить, замазать, уйти, использовать белый проход с ценой.
4. Aftermath: песок в кармане, засветка фото, молчащий свидетель.

## Словарь

"белое окно", "занавесь", "песок", "сухо", "фото", "засветка", "свидетель", "не двор", "область", "замазать".

## Примеры строк

- "Белое окно не открывай. Там не двор."
- "Занавеску держи двумя руками. Свет тянет за взгляд."
- "Песок в кармане сухой, хотя пола там не было."
- "Фотография засветилась. На ней комната, где тебя ещё не было."
- "Короткий белый проход экономит шаги и людей."

## Игровая задача

Веретар должен давать зрительный survival horror и моральный выбор: спасти свидетеля или использовать путь.

## DoD

- 20+ Veretar lines.
- Минимум 5 player-action aftermath rumors.
- Нет "светлого мистицизма".
- После systems changes `npm run check`.
